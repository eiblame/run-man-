# 🏃 RUN MAN

Projeto de jogo inspirado em plataformas clássicas, focado em movimentação, desafios e evolução contínua do gameplay.

⚡ Desenvolvido para praticar:
- Programação de jogos
- Lógica de movimentação
- Estruturação de código
- Sistemas interativos
- Versionamento com Git e GitHub

🎮 O objetivo do projeto é criar uma experiência dinâmica enquanto evoluo minhas habilidades como desenvolvedor.

### 🚀 Tecnologias e conceitos explorados
- Game Development
- Git/GitHub
- Programação orientada à lógica
- Estruturação de projetos
- Mecânicas de plataforma

🌑 Projeto em constante evolução.

> “Run. Survive. Upgrade.”
